# EDA-sQuad-Indexing-Metric-Spaces-for-Exact-Similarity-Search

## Run Backend
To run the backend server, navigate to the `backend` directory and execute the following command:

```bash
npm init -y # Initialize a new Node.js project (if not already done)
npm install express cors # Install dependencies
node server.js
```
## Run Frontend
To run the frontend application, navigate to the `frontend` directory and execute the following command:
```bash
npm i # Install dependencies
npm run dev
```
This will start the frontend application, which can be accessed via a web browser at `http://localhost:5173`.
